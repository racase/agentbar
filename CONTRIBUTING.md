# Contributing

## Development

```powershell
npm install
npm run dev
```

For a production-style local run:

```powershell
npm run build
npx .
```

## Scope

Please keep changes focused. AgentBar is a tray utility, so small and testable changes are preferred over large refactors.

## Pull requests

- Describe the user-visible change clearly.
- Include screenshots for UI changes when relevant.
- Mention any Windows-specific behavior or tray limitations.
- Avoid committing local app state, cookies, session files, or build artifacts.
