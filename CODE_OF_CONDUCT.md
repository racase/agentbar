# Code of Conduct

## Our standard

AgentBar is an open source project. We expect contributors and participants to keep discussion constructive, technical, and respectful.

## Expected behavior

- Be respectful in disagreements.
- Focus feedback on the code, behavior, or design, not on the person.
- Assume good intent, but be precise when raising problems.
- Prefer clear, reproducible reports over vague complaints.

## Unacceptable behavior

- Harassment, insults, personal attacks, or discriminatory language
- Trolling, intimidation, or repeated bad-faith disruption
- Publishing private information without permission
- Sexualized language or unwelcome sexual attention
- Any conduct that would make participation unsafe or unreasonable

## Scope

This applies to repository discussions, issues, pull requests, reviews, and other project communication spaces.

## Enforcement

Project maintainers may edit, remove, or moderate content that violates this code of conduct. Repeated or severe violations may lead to temporary or permanent exclusion from the project spaces.

## Reporting

If you need to report a problem, contact the maintainer privately through GitHub rather than escalating the issue in public.
